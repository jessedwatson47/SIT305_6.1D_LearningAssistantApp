# Learning Assistant
An Android app made to enhance users learning by generating quizzes and feedback powered by AI.


## Tech used
- Android Studio
- GPay
- OpenAI
- Room Database

## Features
- Users can select from a set of topics, AI generates quizzes on these topics.
- Users receive feedback on why they were correct, or how they were wrong after a quiz.
- History is displayed via a RecyclerView so users can see their answered questions alongside their answer and the correct answer.
- Users can share their profile statistics via Androids built in SEND action. Current implementation is just a string.


## Future Improvements
- Expand upon the current quiz generation, allowing the AI to reflect on users aptitude in a given subject and create increasingly more difficult questions to match.
- Add roles for users, with GPay out of test environment.
- Allow sharing via QR code, expand upon the statistics that are currently shared. E.g. ranking if implemented, strongest topic, weakest topic, aspiring job role.
